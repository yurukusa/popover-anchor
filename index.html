Basic Anchor + Popover

Combine anchor-name on a trigger element with position-anchor on a popover to create positioned UI that follows its anchor automatically.

Live Demo anchor-name + popover

Anchored Popover

This popover is positioned using CSS Anchor Positioning. It automatically stays anchored to the button above.

/* Anchor element */ .trigger { anchor-name: --my-anchor; } /* Positioned popover */ [popover] { position-anchor: --my-anchor; position: fixed; top: anchor(bottom); left: anchor(center); translate: -50% 8px; }

How It Works

1. Name the anchor
anchor-name: --my-anchor registers an element as a positioning anchor. The name uses custom property syntax (--prefix).

2. Reference the anchor
position-anchor: --my-anchor on the positioned element links it to the named anchor.

3. Use anchor() function
top: anchor(bottom) positions the element's top edge at the anchor's bottom edge. Available sides: top, bottom, left, right, center.

4. Popover API
The popover attribute provides light-dismiss, top-layer promotion, and toggle behavior for free.

Key Benefits:

  • No JavaScript positioning libraries needed
  • Automatic top-layer (no z-index wars)
  • Light dismiss closes on outside click
  • Browser handles viewport clamping

Position Try Fallbacks

position-try-fallbacks lets the browser automatically flip or reposition the anchored element when it would overflow the viewport. No collision detection JavaScript needed.

Viewport-Aware Positioning position-try-fallbacks

Drag the button to edges of the demo area. The popover will flip to stay visible.

Auto-positioned

This popover flips when near viewport edges using position-try-fallbacks.

.trigger { anchor-name: --anchor; } .tooltip { position-anchor: --anchor; position: fixed; bottom: anchor(top); left: anchor(center); translate: -50% -8px; /* Flip to bottom when too close to top edge */ position-try-fallbacks: flip-block; } /* Available fallback keywords: flip-block — flip in block axis (top↔bottom) flip-inline — flip in inline axis (left↔right) flip-start — flip start edges Both can be combined: position-try-fallbacks: flip-block flip-inline; */

CSS Anchor Positioning — Complete Property Reference

Property / Function Values Description
anchor-name none | <dashed-ident>+ Registers an element as a positioning anchor. Must use -- prefix.
position-anchor auto | <dashed-ident> Sets the default anchor for a positioned element.
anchor() anchor(<side>, <fallback>?) Returns the position of an anchor's edge. Sides: top, bottom, left, right, center, start, end.
anchor-size() anchor-size(width | height | block | inline) Returns the size of the anchor element in a given dimension.
position-area <block> <inline> Grid-based positioning: top/center/bottom x left/center/right.
position-try-fallbacks flip-block | flip-inline | flip-start | <dashed-ident> Auto-flip when positioned element overflows viewport.
position-try-order normal | <try-size> Determines the order in which fallback positions are tried.
position-visibility always | anchors-visible | no-overflow Controls visibility based on anchor visibility or overflow.
@position-try @position-try --name { ... } Defines a named fallback position with custom positioning rules.

Popover API — Property Reference

HTML Attributes

popover or popover="auto" — Auto-dismiss on outside click or Escape.
popover="manual" — Must be dismissed programmatically. No light dismiss.
popovertarget="id" — Links a button to toggle a popover by ID.
popovertargetaction — "toggle" (default), "show", or "hide".

JavaScript API

element.showPopover() — Opens the popover.
element.hidePopover() — Closes the popover.
element.togglePopover() — Toggles open/close state.
'toggle' event — Fires when popover opens or closes.
'beforetoggle' event — Fires before state change. Cancelable.

CSS Pseudo-elements for Popover

::backdrop

Full-screen overlay behind top-layer elements. Style it for modal-like dimming.

:popover-open

Matches when the popover is in the open state. Use for entry animations.

Define initial styles for popover entry transitions. Enables CSS-only open animations.
